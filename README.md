# Економічна безпека – Левецька (українська версія з графіками)

Навчальний мінісайт, створений для демонстрації:

- особистого резюме студентки Олександри Левецької;
- узагальненої характеристики економічної безпеки Нідерландів;
- основних загроз економічній безпеці та можливих напрямів протидії;
- візуалізації ключових показників економічної безпеки за допомогою графіків (Chart.js).

## Структура проєкту

```text
ekonomichna-bezpeka-levetska-uk-graphs/
├─ index.html          # Сторінка «Резюме»
├─ security.html       # «Економічна безпека країни» + графіки
├─ threats.html        # «Загрози та протидія»
├─ styles/
│   └─ main.css        # Основні стилі сайту (пастельний дизайн)
└─ images/
    └─ profile.jpg     # Фото для резюме
```

## Як запустити локально

1. Розпакуйте архів у будь-яку папку.
2. Відкрийте файл `index.html` у браузері.

## Як опублікувати на GitHub Pages

1. Створіть новий репозиторій на GitHub.
2. Завантажте в нього **усі файли** з цієї папки (`index.html`, `security.html`, `threats.html`, папку `styles`, папку `images`).
3. У налаштуваннях репозиторію (Settings → Pages):
   - Оберіть гілку `main` (або `master`) і теку `/root`.
4. Збережіть — через деякий час GitHub надасть посилання на опублікований сайт.
